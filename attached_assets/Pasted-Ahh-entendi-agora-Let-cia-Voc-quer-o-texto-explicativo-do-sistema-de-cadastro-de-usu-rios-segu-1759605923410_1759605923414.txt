🧾 SISTEMA DE CADASTRO DE USUÁRIOS

💡 Explicação completa do funcionamento

🎯 Objetivo do Programa

O objetivo deste programa é desenvolver um sistema de cadastro de usuários, que evolui em várias etapas — do mais simples ao mais completo.
A atividade tem como finalidade praticar os principais conceitos de lógica de programação, como declaração de variáveis, entrada e saída de dados, estruturas de decisão, laços de repetição, vetores (listas) e menus interativos.
Ao final, o sistema deve permitir cadastrar usuários, listar todos os cadastros, procurar nomes específicos e encerrar o programa.


---

🧩 Etapa 1 – Entrada e Exibição de Dados

Na primeira fase, o programa pede duas informações básicas ao usuário:

Seu nome

Sua idade


Após a digitação, o programa exibe uma mensagem personalizada de boas-vindas, mostrando os dados informados.

Exemplo de execução:

Digite seu nome: Maria
Digite sua idade: 20
Olá, Maria! Você tem 20 anos.

📘 Conceitos aprendidos:

Criação e uso de variáveis para armazenar informações.

Utilização do comando de entrada de dados (input()).

Exibição de mensagens personalizadas com print().


Essa primeira etapa ensina como coletar e apresentar dados simples ao usuário, base para as próximas fases.


---

🧩 Etapa 2 – Criação do Menu Interativo

Na segunda etapa, o programa evolui e passa a conter um menu principal, onde o usuário pode escolher o que deseja fazer.
As opções apresentadas são:

1. Cadastrar usuário


2. Mostrar usuário cadastrado


3. Sair



O programa repete o menu até que o usuário escolha a opção “Sair”.

📘 Conceitos aprendidos:

Uso de estruturas de repetição (while) para manter o programa ativo.

Aplicação de estruturas de decisão (if, elif, else) para definir o que será executado conforme a escolha do usuário.

Noção de menu interativo, permitindo várias ações dentro de um mesmo sistema.



---

🧩 Etapa 3 – Ampliação para Múltiplos Cadastros

A terceira fase amplia o sistema, permitindo cadastrar vários usuários.
Para isso, é criada uma lista (vetor) onde cada cadastro é armazenado.
Cada usuário tem nome e idade, e o sistema guarda essas informações para serem usadas depois.

Também é definido um limite máximo de 5 usuários cadastrados.
Caso o limite seja atingido, o programa mostra uma mensagem de aviso e não permite novos cadastros.

📘 Conceitos aprendidos:

Utilização de listas para armazenar vários elementos
